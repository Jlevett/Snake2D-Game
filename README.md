# 2D Snake Game

## Introduction
This is a simple 2D snake Game

![gif](https://github.com/Jlevett/Snake2D-Game/blob/master/snake.gif)

## Installation
1. Via command prompt locate bin directory
2. Type `java Main` 
3. Have fun and _enjoy_
